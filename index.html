Desarrollé piezas gráficas, sliders, banners, GIFs, presentaciones gráficas, plantillas personalizadas y un memory game interactivo para OnStar, adaptándolos a su plataforma de e-learning. Cada diseño fue cuidadosamente alineado con su identidad de marca, enfocándome en crear elementos visualmente atractivos y funcionales. Los banners y sliders destacaron contenido clave con un diseño dinámico, mientras que las plantillas y presentaciones facilitaron una comunicación clara y profesional. Además, el memory game interactivo ofreció una experiencia educativa entretenida y efectiva. Este proyecto combinó creatividad y estrategia, optimizando la experiencia de aprendizaje de los usuarios.
